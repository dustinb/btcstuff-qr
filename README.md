# Quick Start

Build and run the UI and api

```bash
docker compose up
```

Browse to http://localhost:8101/ for the UI. The API responds to request for QR codes at /qr

http://localhost:8101/qr?address=bc1qkd5az2ml7dk5j5h672yhxmhmxe9tuf97j39fm6
